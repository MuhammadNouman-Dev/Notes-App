# 📝 Notes App (React)

This is a simple Notes App built using **React.js**. It allows users to create, view, and delete notes. The notes are stored in **local storage**, so they remain saved even after refreshing the page.

---

## 🚀 Features

- Add new notes
- Delete existing notes
- Notes persist using **local storage**
- Responsive UI using **Tailwind CSS**
- Smooth state management using React Hooks

---

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6+)
- useState Hook
- useEffect Hook
- Tailwind CSS
- Local Storage (Browser API)

---

## 📸 Project Description
 
It is a simple and beginner-friendly React project that demonstrates how to manage state, handle user input, and store data locally in the browser.

---

## 📌 How It Works

1. User writes a note in the input field  
2. Clicks the "Add" button  
3. Note is saved and displayed on the screen  
4. User can delete notes anytime  
5. All notes are stored in local storage, so they don’t disappear after refresh

---

## 💡 Purpose

The purpose of this project is to practice:
- React Hooks (useState, useEffect)
- Component-based architecture
- Working with local storage
- Styling with Tailwind CSS
